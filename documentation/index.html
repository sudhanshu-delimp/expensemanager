Quick Start Guide

 
A simple but powerful PHP script to manage your expenses with multi-user level and permission. Our expense management system designed to help individual or business budget, track and possibly control your expenses. It supports tracking of both your expenses and income. This expense management system provides an integrated set of features to help you to manage your expenses and cash flow. It provides the ability to group your income/expenses into categories and lets you set a budget and track expenses in the category.
 

Features

  • Manage Expense
  • Manage Income
  • Manage Expense Category
  • Manage Income Category
  • Filter by date range
  • Sorting & Pagination
  • Support Multi-user level to Manage the system
  • User Login & Management Included
  • Set different permission for different user type
  • Custom CRUD can be added
  • Easy to customize

 

Installation

Server Requirements
  • Apache with PHP and MySQL server
  • PHP v5.5+
  • MySqli
  • GD Extension
  • CURL Enabled
  • Openssl Enabled
Quick Installation Guide
  • Unzip the expense_management.zip which you have downloaded from the Envato Market.
  • Upload the files to your sever using any FTP client.
  • The file structure should be look like this:
    • /application
    • /assets
    • /documentation
    • /files
    • /install
    • /system
    • /index.php
  • Create a MySql Database via your web hosting providers Control Panel (e.g. cPanel). Once you have created the database, remember the following information:
    • Database Name
    • Database Username
    • Database Password
  • Open your web browser and enter the url (http://yourdomain.com or http://yourdomain.com/foldername) in the address bar.
  • If everything is ok, you should get the installation page.
  • Follow the instructions and click on the Next button. Then enter the database details which you have collected earlier. Finally, enter your name, email and password which will be required to login in the application. Then click on the Finish button and wait a while. The system will create necessary tables in database, files and configuration.
  • That's all! After completing the installation delete the installation folder and login to your dashboard.

 

Dashboard

You can see analytical data on your dashboard 

User's Analytics
  1. Total Users: Number of users registered on the site.
  2. Today Registered: Number of users registered on current date.
  3. Active Users: Number of users who's account status is "Active" on the site.
Last Registrations
  1. List of user's who are registered recently. It will show list of upto 10 users max.
  2.  You can define different currency for different clients. It's useful when you are working with a foreign client which currency symbol is different from your default currency symbol. By default the invoices of the client will be generated using the default currency symbol if you don't define any currency symbol with that specific client.
Monthly Registrations
  1. Shows bar graph on monthly basis, with number of registered users for each month. Each Bar represent the number of user registered in particular month

Application Structure

  1. The config folder contains all configuration related files.
  2. The controllers folder contains all business logic of the application.
  3. Views folder contains all the view related files.
  4. Assets folder contains all css, js and images of the application.
  5. Don't change anything in system folder. It's contains the core functions of the application.
